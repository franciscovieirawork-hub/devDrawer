# DevDrawer

Interactive whiteboard and planner for developers. Plan, draw, sketch and organize your software projects.

## Tech Stack

- **Frontend**: React + Next.js (App Router)
- **Backend**: Next.js API Routes (TypeScript)
- **Database**: PostgreSQL
- **ORM**: Prisma 7

## Getting Started

### Prerequisites

- Node.js 20+
- Docker Desktop
- npm

### 1. Install dependencies

npm install
